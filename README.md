# AutoClicker for macOS

A lightweight open source macOS autoclicker inspired by Speed AutoClicker.

## Installation

Download the latest release DMG and drag the app to Applications.

Or install with terminal:

curl -fsSL https://raw.githubusercontent.com/59Codings/AutoClicker/main/install.sh | bash

## First Launch

macOS may block the app because it is from an unidentified developer.

Right click the app and press Open.

Or go to:

System Settings
Privacy and Security
Open Anyway

## Accessibility Permission

The app needs permission to control mouse input.

Open:

System Settings
Privacy and Security
Accessibility

Enable AutoClicker.

Restart the app after enabling.

## Features

Global hotkey
Toggle or Hold activation
Clicks per second configuration
Random CPS mode
Unlimited clicking mode
Click duty cycle control
Click limit system
Mouse button selection
Language switching
Start with macOS option
Automatic settings saving
Reset settings button

## Building

Open the project in Xcode and press Run.
